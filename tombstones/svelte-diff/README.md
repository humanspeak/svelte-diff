# svelte-diff

This is an installable alias for **[@humanspeak/svelte-diff](https://www.npmjs.com/package/@humanspeak/svelte-diff)**, the canonical package — a Svelte 5 component for visual string diffing using the diff-match-patch algorithm.

`npm install svelte-diff` works and re-exports the real package, but prefer installing `@humanspeak/svelte-diff` directly.

```bash
npm install @humanspeak/svelte-diff
```

Docs: <https://diff.svelte.page> · Source: <https://github.com/humanspeak/svelte-diff>
